## 这是一个任务清单软件。

***



### v1.0

#### 功能介绍

* 任务清单功能。可以在后台添加任何一天的任务并在清单界面提醒今日任务。
* 事件倒计时。可在后台添加一些重要的事情，并会在清单界面提醒最近三件事情的日期和内容。
* 天气查询。会在清单界面显示实时的天气情况（需联网）。
* 单词功能。可在清单界面随机获取单词，并显示发音和解释、例句和翻译，内置10000+单词。
